## Useful Tools

- [gethomepage](https://github.com/gethomepage/homepage) - 100개 이상의 서비스 연동과 Docker 자동 탐지를 지원하는 커스터마이징 가능한 셀프호스팅 대시보드. `dashboard` `self-hosted`
- [n8n](https://github.com/n8n-io/n8n) - 400개 이상의 서비스를 연결하고 AI 워크플로우를 구축할 수 있는 셀프호스팅 가능한 워크플로우 자동화 플랫폼. `automation` `self-hosted`
- [NotebookLM](https://notebooklm.google.com/) - 업로드한 문서를 기반으로 학습과 리서치를 도와주는 Google의 AI 기반 개인 연구 어시스턴트. `ai` `knowledge-management`
- [it-tools](https://github.com/CorentinTh/it-tools) - 개발자를 위한 다양한 온라인 유틸리티를 모아놓은 웹 도구 모음. `self-hosted`
- [qmd](https://github.com/tobi/qmd) - 마크다운 노트, 문서, 지식 베이스를 로컬에서 검색할 수 있는 하이브리드 검색 엔진 (BM25 + 벡터 + LLM 리랭킹). `cli` `mcp`
